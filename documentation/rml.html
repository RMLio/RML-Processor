RML: A generic Mapping Language Schema for RDF

Purpose

This document defines an RDF Schema, which can be used to specify mappings of data in structured formats to RDF. RML is defined as an extension of R2RML.

Usage

The namespace URI is:
http://mmlab.be/ns/rml#
The preferred prefix is:
rml

Comments

This wok is a first prototype of RML.

Base Source (OWL Class)

Denotes any resource. Currently limited to structured data formats: a database table, an XML file, a CSV file, a JSON file or an RML view

Object Properties

logical Source (Property)

Represents the logical source to be mapped. This can be a pointer to any dataset.

Definition

The URI of this property is http://mmlab.be/ns/rml#logicalSource

This property is an object property.

The domain of this property is Triples Map

query Language (Property)

It defines the query language used or the language used to refer to the elements of the source file.

Definition

The URI of this property is http://mmlab.be/ns/rml#queryLanguage

This property is an object property.

The domain of this property is Logical Source
The range of this property is Query Language

Data Properties

selector (Property)

query (Property)

A valid (considering the specified language) query.

Definition

The URI of this property is http://mmlab.be/ns/rml#query

This property is a data property.

The domain of this property is Logical Source
The range of this property is string.

version (Property)

It defines the version of the query language used.

Definition

The URI of this property is http://mmlab.be/ns/rml#version

This property is a data property.

The domain of this property is RML View

source Name (Property)

Schema-qualified name of a source (e.g., the filename, valid URL).

This property is a data property.
